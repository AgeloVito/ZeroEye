Usage: ZeroEye [options]
    -h     帮助
    -i    <Exe 路径>    列出Exe的导入表
    -p    <文件路径>    自动搜索文件路径下可劫持利用的白名单

example：
  ZeroEye.exe -p c:\             //搜索c盘所有exe是否有劫持的可能
  ZeroEye.exe -i aaa.exe         //判断指定exe是否有劫持的可能
